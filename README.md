# background_mask_with_image  
下記自作スクリプトに、画像を読み込む機能を追加した。  
https://github.com/AKIRA1921626/background_mask  
背景つきマスクに画像も読み込めるように機能を追加した。トラックバーも追加した。  
Aviutl2用のスクリプトとなります。
 
二次配布可、改変可、クレジット不要。  
悪例として、批判目的での利用、これらも可能となります。  

ASU氏が作成した、多機能上位互換版がある。
